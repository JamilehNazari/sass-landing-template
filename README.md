## About the Project

This is a clean, single-page template built to showcase professional frontend development skills. The focus of this project is on **SCSS architecture** and **modern workflow automation using Gulp**, providing a strong example of structured styling, modular code, and efficient build processes.

It is designed as a **portfolio-ready project** for developers seeking to demonstrate their technical proficiency in HTML, SCSS, Bootstrap, and Gulp.

---

## Technologies Used

- **HTML5** – Structure of the webpage  
- **SCSS (Sass)** – Modular and maintainable CSS workflow  
- **Bootstrap** – Utility classes and layout support  
- **Gulp** – Automation tasks (compiling SCSS, minifying CSS, etc.)  
- **AOS** – Scroll animations  
- **NPM** – Dependency management
